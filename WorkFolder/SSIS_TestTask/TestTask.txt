На вход система ожидает файлы 2-х типов (примеры файлов прилагаются).
Файлы всегда плоские, формата csv.

1) Stage: создать схему, создать таблицы, соответствующие входным файлам.

2) Создать таблицу для логирования действий с данными (имя файла, дата\время загрузки, кол-во строк в файле, дата\время обработки, кол-во строк попавших в хранилище)

3) Создать в SSIS процесс, который забирает файлы из папки и загружает их в таблицы в Stage.
   Пакет (или проект) SSIS должен иметь переменные, в которых можно указать:
		- путь к папке, откуда система забирает файлы
		- сервер\имя БД куда система будет заливать данные
		- логин\пароль для пользователя БД (пароль должен быть Sensitive)

3) DWH: Проанализировать природу данных и создать таблицу фактов и минимум 4 дименшена (в схеме DWH) 
   Написать SSIS пакет\проект (или дополнить созданный на шаге (3)), который забирет данные со Stage и загружает из в DWH.
		- при этом дименшены должны автоматически заполняться данными
		- таблица фактов должны иметь ссылки на заполненные данные дименшенов. 		
		
4) Как результат тестового задания предоставить набор скриптов, которые позволят создать БД, схемы, таблицы (любые другие объекты БД, которые были созданы при 
   выполнении задания) + SSIS пакет\проект + скрипт для создания environment variables, для задания значений переменным пакета\проекта.
